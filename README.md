# Training & Placement Cell

A full-stack web application developed to manage the Training and Placement Cell activities of a college. The system allows Admins, Companies, and Students to manage placement-related operations through a single platform.

## Features

- User Registration and Login
- Role-based access (Admin, Company, Student)
- Student Management
- Company Management
- Job Posting Management
- Job Applications
- Placement Records
- Notice Management
- Student Profile
- Company Profile

## Tech Stack

### Frontend
- React.js
- JavaScript
- HTML
- CSS
- Axios

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Maven

### Database
- MySQL

## Installation

### Clone the repository

```bash
git clone https://github.com/matre-pratik/training-placement-cell.git
```

### Backend

```bash
cd placement-cell-backend
mvn spring-boot:run
```

### Frontend

```bash
cd placement-cell-frontend
npm install
npm run dev
```

## Modules

### Admin
- Manage Students
- Manage Companies
- Manage Job Postings
- Manage Applications
- Manage Placement Records
- Manage Notices

### Company
- Manage Company Profile
- Create and Manage Job Postings
- View Applications

### Student
